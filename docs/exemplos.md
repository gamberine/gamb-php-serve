# Exemplos

## Primeiro uso

```bash
cd /d/ProjetosGit/meu-projeto
gamb-php-serve
```

## Uso com porta customizada

```bash
gamb-php-serve --port 8080
```

## Uso em foreground

```bash
gamb-php-serve --foreground
```

## Verificação

```bash
gamb-php-check
```

## Status

```bash
gamb-php-status
gamb-php-status --all
```

## Parada

```bash
gamb-php-stop
gamb-php-stop --all
```

## Remoção

```bash
gamb-php-remove
gamb-php-remove --with-logs
```
